Requirements
------------

To work correctly, you will first need:

* `python`_ (v3.8 or recent) must be installed.
* `pip`_ (v20.0 or recent) must be installed.

Installing
----------

Globally:

.. code-block:: shell

    $ sudo pip install zeroed

For the user:

.. code-block:: shell

    $ pip install zeroed --user
